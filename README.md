# Modern Realtime Chat Application

This is a full-stack realtime chat application built with a monorepo structure.

## Tech Stack

### Backend
- **Framework**: NestJS
- **Language**: TypeScript
- **Database**: MySQL (via Prisma ORM)
- **Authentication**: JWT (Passport)
- **Realtime**: Socket.IO

### Frontend
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: React Hooks

## Project Structure

- `backend/`: NestJS API server application
- `frontend/`: Next.js frontend application
- `docs/`: Project documentation and design documents

## Getting Started

See the README files in each subdirectory for specific setup instructions.
